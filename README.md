# MeteoApp

MeteoApp — это простое приложение Windows Forms для отображения текущей погоды по выбранному городу с использованием API OpenWeatherMap.

## Что умеет приложение

- Показывает текущую температуру и описание погоды;
- Загружает иконку погоды;
- Поддерживает выбор города из списка;
- Использует API OpenWeatherMap для получения данных;
- Позволяет пользователю ввести и сохранить API key через настройки приложения.

## Требования

- .NET Framework 4.7.2;
- Visual Studio 2017+ или MSBuild;
- Доступ к сети для запросов к API.

## Установка и запуск

1. Откройте решение MeteoApp.sln.
2. Восстановите NuGet-пакеты (в проекте используется Newtonsoft.Json).
3. Соберите проект и запустите его.
4. Нажмите кнопку "Настр." и введите ваш API key OpenWeatherMap.
5. Ключ будет сохранён в пользовательских настройках приложения и будет использоваться при следующих запусках.

## Структура проекта

- MainForm.cs — основная форма приложения и логика работы с API;
- MainForm.Designer.cs — визуальный интерфейс формы;
- Weather.cs — модели данных ответа API;
- Program.cs — точка входа приложения;
- Properties/Settings.settings и Properties/Settings.Designer.cs — пользовательские настройки, включая API key и язык;
- packages.config — список зависимостей NuGet.

## Использование

- Выберите город из списка;
- Нажмите кнопку для обновления данных о погоде;
- При необходимости нажмите "Настр." и измените API key;
- Выберите язык через меню "Language" в верхней части окна.

## Примечание

Для работы приложения необходим действительный API-ключ OpenWeatherMap. Без него приложение покажет сообщение о необходимости ввести ключ в настройках.
